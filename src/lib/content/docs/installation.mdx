---
title: Installation
description: How to install dependencies and structure your app.
---

For shadcn/ui installation, visit the <a href="https://ui.shadcn.com/docs/installation" target="_blank" rel="noopener noreferrer">shadcn/ui docs</a>.

To use the components in your React app, initialize shadcn/ui, copy the `:root`, `.dark`, and `@theme inline` styles from `src/app/globals.css` to your app's globals.css.

You can then add components from this registry by running the `npx shadcn@latest add` command from each component's docs page.
